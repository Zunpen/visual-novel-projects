# Visual Novel Project
Repo project membuat visual novel untuk matkul KWN

## Structure
- trial/ → short project untuk belajar Ren'Py
- main-game/ → final project

## Engine
Ren'Py

## How to run
1. Open Ren'Py Launcher
2. Select project
3. Click "Launch Project"
