# 🔄 Workflow

1. Writer → membuat cerita  
2. Artist → membuat aset  
3. Programmer → implementasi  
4. QA → testing  
5. Ulangi proses

## Overview
- trial/ → short project untuk belajar Ren'Py
- main-game/ → final project

# Panduan Singkat tiap role
# 🚀 Aturan Umum
- Hati-hati jangan mengubah bagian role lain
- **Jangan lupa gunakan branch dev** (pull dan push dilakukan di branch dev)
- Kalo bingung tanya aja gaes

## 🧠 Narrative & Game Design
- Menulis cerita di `docs/story.md`
- Mendefinisikan karakter di `docs/characters.md`
- Merancang konsep di `docs/concept.md`

## 💻 Programmer
- Mengimplementasikan script ke Ren’Py
- Mengatur:
  - Alur scene
  - Gambar
  - Audio
  - Menjaga struktur proyek

- Bekerja di:
  - `game/`
  - `script.rpy`
  - `characters.rpy`


**Note:** Hanya programmer yang mengubah struktur inti

## 🎨 Creative Team

### 🖌️ Illustrator
- Membuat karakter & background

Upload ke:
- `assets/characters/`
- `assets/backgrounds/`

---
### 🖥️ UI/UX Designer
- Mendesain UI, menu, dan tombol

Upload ke `assets/UI`

---
### 🎧 Composer
- Membuat musik & sound effect

Upload ke `assets/music/`

---
## 🧪 QA Tester
- Menguji game
- Melaporkan:
  - Bug
  - Typo
  - Pilihan yang tidak berfungsi

**Format laporan:**
- Apa yang terjadi
- Di mana terjadi
- Screenshot (opsional)

---

## Engine
Ren'Py

## How to run
1. Open Ren'Py Launcher
2. Select project
3. Click "Launch Project"
